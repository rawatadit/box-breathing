# Box Breathing App

A calming box breathing app with smooth animations to help you relax and focus.

## Features

- 4-phase breathing cycle (Inhale → Hold → Exhale → Hold)
- Visual box animation with glowing tracer
- Audio cues for each phase
- Multiple session types (Quick Calm, Deep Relaxation, Focus, Custom)
- Pause/Resume functionality
- Keyboard shortcuts (Space: Start/Pause, Escape: Back)

## Usage

Open `index.html` in your browser - no server required.

## How Box Breathing Works

1. **Inhale** for 4 seconds
2. **Hold** for 4 seconds
3. **Exhale** for 4 seconds
4. **Hold** for 4 seconds

Repeat to promote relaxation and mental clarity.
